# AliyunDDNS
阿里云动态DNS

# 使用方法
```
Usage: AliyunDDNS      [OPTION]
  -c, --config config_file      加载配置文件
  -r, --region region_id        地区ID
  -i, --key-id key_id           密钥ID
  -s, --key-secret key_secret   密钥Secret
  -d, --domain domain           域名
  -t, --host host               host记录
  -h, --help                    打印帮助
```
